# Distributor Management System - Products Page

A React-based products management page with full CRUD functionality, pagination, and filtering.

## Features

- ✅ Display products in a table with all details (Image, Name, Category, MRP, Distributor Rate, Retailer Price, UOM, CRT)
- ✅ Add new products via modal form
- ✅ Edit existing products
- ✅ Delete products with confirmation
- ✅ Category filter dropdown
- ✅ Pagination (10 items per page)
- ✅ Responsive design

## Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to `http://localhost:3000`

## Usage

- **Filter Products**: Use the category dropdown at the top to filter products by category (All, Milk, Butter Milk, Dahi)
- **Add Product**: Click the "+ Add Product" button to open the modal and fill in the product details
- **Edit Product**: Click the "Edit" button on any product row to modify its details
- **Delete Product**: Click the "Delete" button and confirm to remove a product
- **Navigate Pages**: Use the pagination controls at the bottom to browse through products

## Technologies Used

- React 18
- Vite
- CSS3
